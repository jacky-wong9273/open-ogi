# Tools

## Git Operations

**Reference**: `labs/tools/git-operations/TOOL.md`
Provides git version control operations including clone, branch, commit, push, pull, diff, and log. The Tech Lead uses this tool to review diffs across branches, inspect commit history for architectural changes, and manage repository operations during critical incident resolution.

## Code Analyzer

**Reference**: `labs/tools/code-analyzer/TOOL.md`
Runs static analysis, linting, type checking, complexity analysis, and dependency auditing on source files. The Tech Lead uses this tool to enforce code quality standards, evaluate cyclomatic complexity against thresholds, audit dependencies for vulnerabilities, and generate quality reports for merge policy decisions.

## Jira Integration

**Reference**: `labs/tools/jira-integration/TOOL.md`
Interacts with Jira for issue tracking and sprint management, supporting issue creation, updates, transitions, JQL queries, and comments. The Tech Lead uses this tool to track technical debt, manage architecture decision records, estimate technical complexity during planning, and coordinate on cross-cutting technical concerns.

## Telegram Messenger

**Reference**: `labs/tools/telegram-messenger/TOOL.md`
Sends and receives messages via the Telegram Bot API for team communication. The Tech Lead uses this tool to provide architecture guidance to developers, communicate technical decisions, and coordinate with the team on complex implementations and production incident response.
