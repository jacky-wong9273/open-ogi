# Skills

## Code Review

**Reference**: `labs/skills/code-review/SKILL.md`
Performs structured code reviews on diffs and pull requests. The Tech Lead uses this skill to conduct and coordinate code reviews across all pull requests, evaluating architecture decisions, security concerns, performance implications, and adherence to coding standards.

## Sprint Planning

**Reference**: `labs/skills/sprint-planning/SKILL.md`
Facilitates sprint planning sessions including story estimation, capacity calculation, and sprint goal definition. The Tech Lead uses this skill to assess technical complexity of backlog items, estimate effort using a Fibonacci scale, and flag stories with high uncertainty or architectural dependencies.

## Bug Triage

**Reference**: `labs/skills/bug-triage/SKILL.md`
Classifies, prioritises, and assigns incoming bug reports. The Tech Lead uses this skill to investigate critical defects, identify root causes from a technical perspective, assign bugs to the appropriate developer based on component ownership, and escalate security concerns.
