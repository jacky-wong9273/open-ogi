# Tools

## Git Operations

**Reference**: `labs/tools/git-operations/TOOL.md`
Provides git version control operations including clone, branch, commit, push, pull, diff, and log. The Developer uses this tool to create feature branches, commit atomic changes, push pull requests, and retrieve diffs for code review.

## Code Analyzer

**Reference**: `labs/tools/code-analyzer/TOOL.md`
Runs static analysis, linting, type checking, complexity analysis, and dependency auditing on source files. The Developer uses this tool to ensure code quality, catch lint errors, verify type safety, and generate quality reports before submitting pull requests.

## Jira Integration

**Reference**: `labs/tools/jira-integration/TOOL.md`
Interacts with Jira for issue tracking and sprint management, supporting issue creation, updates, transitions, JQL queries, and comments. The Developer uses this tool to update story statuses, log work, transition tickets through workflow stages, and communicate progress on assigned tasks.

## Telegram Messenger

**Reference**: `labs/tools/telegram-messenger/TOOL.md`
Sends and receives messages via the Telegram Bot API for team communication. The Developer uses this tool to report blockers, coordinate with teammates, and share updates with the team during the sprint.
