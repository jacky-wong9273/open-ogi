# Skills

## Code Review

**Reference**: `labs/skills/code-review/SKILL.md`
Performs structured code reviews on diffs and pull requests. The Developer uses this skill to review peers' code, evaluating correctness, security, performance, maintainability, and test coverage, producing a checklist with severity-classified findings and an approval verdict.

## Bug Triage

**Reference**: `labs/skills/bug-triage/SKILL.md`
Classifies, prioritises, and assigns incoming bug reports. The Developer uses this skill to validate bug reports, assess severity (P0-P3), identify the affected component, and recommend immediate actions for bugs encountered during implementation and testing.
