# Tools

## Jira Integration

**Reference**: `labs/tools/jira-integration/TOOL.md`
Interacts with Jira for issue tracking and sprint management, supporting issue creation, updates, transitions, JQL queries, and comments. The Product Owner uses this tool to manage the product backlog, create user stories with acceptance criteria, prioritise issues, and track sprint progress.

## Telegram Messenger

**Reference**: `labs/tools/telegram-messenger/TOOL.md`
Sends and receives messages via the Telegram Bot API for team communication. The Product Owner uses this tool to communicate with the development team, answer questions about acceptance criteria, and coordinate on sprint scope decisions.

## WhatsApp Messenger

**Reference**: `labs/tools/whatsapp-messenger/TOOL.md`
Sends and receives messages via the WhatsApp Business Cloud API for stakeholder communication. The Product Owner uses this tool to keep stakeholders informed with high-level product updates, share sprint outcomes, and gather feedback from external parties.
