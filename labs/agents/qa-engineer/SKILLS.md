# Skills

## Bug Triage

**Reference**: `labs/skills/bug-triage/SKILL.md`
Classifies, prioritises, and assigns incoming bug reports. The QA Engineer uses this skill to validate reported defects, classify severity based on impact and urgency (P0-P3), and recommend targeted next steps for resolution.

## Code Review

**Reference**: `labs/skills/code-review/SKILL.md`
Performs structured code reviews on diffs and pull requests. The QA Engineer uses this skill to review code changes from a quality perspective, verifying that tests cover edge cases, error paths are handled, and no flaky test patterns are introduced.
