# Tools

## Jira Integration

**Reference**: `labs/tools/jira-integration/TOOL.md`
Interacts with Jira for issue tracking and sprint management, supporting issue creation, updates, transitions, JQL queries, and comments. The QA Engineer uses this tool to create and track bug reports with reproduction steps, update defect statuses, and query issues for regression testing.

## Code Analyzer

**Reference**: `labs/tools/code-analyzer/TOOL.md`
Runs static analysis, linting, type checking, complexity analysis, and dependency auditing on source files. The QA Engineer uses this tool to identify code quality issues, verify test coverage thresholds, and audit dependencies for known vulnerabilities before release sign-off.

## Telegram Messenger

**Reference**: `labs/tools/telegram-messenger/TOOL.md`
Sends and receives messages via the Telegram Bot API for team communication. The QA Engineer uses this tool to report test results, communicate quality metrics to the team, and coordinate with developers on bug verification and edge case clarification.
