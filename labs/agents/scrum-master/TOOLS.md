# Tools

## Telegram Messenger

**Reference**: `labs/tools/telegram-messenger/TOOL.md`
Sends and receives messages via the Telegram Bot API for team communication. The Scrum Master uses this tool to send daily stand-up summaries, share sprint metrics with the team, and coordinate on blockers and process updates.

## WhatsApp Messenger

**Reference**: `labs/tools/whatsapp-messenger/TOOL.md`
Sends and receives messages via the WhatsApp Business Cloud API for stakeholder communication. The Scrum Master uses this tool to report sprint health to stakeholders, share velocity and burndown metrics, and communicate escalated blockers to external parties.

## Jira Integration

**Reference**: `labs/tools/jira-integration/TOOL.md`
Interacts with Jira for issue tracking and sprint management, supporting issue creation, updates, transitions, JQL queries, and comments. The Scrum Master uses this tool to track sprint progress, query the active sprint board, monitor story transitions, and maintain sprint metrics like velocity and burndown.
